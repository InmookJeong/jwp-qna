## 작성 유의사항
1. 일급컬렉션 사용
   - 대상 : Question, Answer
2. Method 길이 10 Line 이하로 작성
3. 모든 원시값을 포장
   - 대상 : Entity의 Columns
4. 클래스의 인스턴스 변수는 2개 이하로 작성
5. 단위 테스트 가능하도록 코드 분리
